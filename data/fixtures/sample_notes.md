# Gradient Descent and NPV Notes

Gradient descent updates model parameters by moving opposite the gradient of the loss function. The common update rule is theta := theta - eta * gradient. A smaller learning rate can improve stability but may slow convergence.

In regression, mean squared error measures the average squared difference between predictions and observed values. Feature scaling often helps gradient descent converge faster.

Net present value discounts each cash flow by the required return. For cash flows C0, C1, and C2, NPV equals C0 + C1 / (1 + r) + C2 / (1 + r)^2.
