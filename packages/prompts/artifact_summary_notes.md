# Summary Notes

Render concise summary notes for the notebook "{{notebook_title}}".

Inputs:
- Key concepts: {{concepts}}
- Source summaries: {{summaries}}

Output a Markdown document with:
- `# Summary Notes for {{notebook_title}}`
- `## Key Ideas` — one bullet per key concept.
- `## Source Summaries` — one bullet per source summary.

Keep it faithful to the sources. Attach citations to any specific claim, formula, or figure.
